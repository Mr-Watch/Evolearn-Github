View Manager

~~~~~~

First you have to select an activity

You can either click on the at the top left of the screen or the in the center

View Controls

When you have an active view you will see this

arrangement of buttons.
You can have multiple view open at any given time.

You can use the buttons to navigate the different currently opened views or use the button to get a window with all the views ordered by the latest opened, so you can click on the one you like without having to repeatably go multiple times forwards or backwards.
